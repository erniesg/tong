# Asset Presets

Store reusable visual presets and generated media manifests here.

Suggested structure:
1. `backgrounds/` city and location backdrops
2. `ui/` chat skins, badges, icons
3. `rewards/` video-call placeholders and polaroid templates
4. `manifest.json` canonical asset index

Naming convention:
- `<city>_<location>_<variant>_<vN>.<ext>`
- example: `seoul_food_street_evening_v1.png`
